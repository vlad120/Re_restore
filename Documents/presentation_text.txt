WebServer project presentation

Я бы хотел презентовать свой проект интернет-магазина, сайта,  с серверной частью, написанной на Python.
<>
Как результат получился простой сайт с интуитивным интерфейсом и оптимизацией под различные устройства. Насчёт оптимизации, у страниц задана минимальная ширина, благодаря чему на дисплеях с вертикальной ориентацией они не превращаются какую-то смесь элементов, а на широких мониторах сайт автоматически расширяет и увеличивает свои элементы.

В принципе, функционала сервиса достаточно, чтобы  запустить настоящий интернет-магазин. А при необходимости в него можно легко внести изменения, или добавить новые функции.

Пользователи могут создавать учетные записи, изменять свои данные, делать заказы, отменять их. Администратор, в свою очередь, может управлять пользователями, заказами, добавлять, редактировать и удалять товары.
<>
Немного о реализации. Скрипт обеспечивает обработку http запросов, отвечая на них сгенерированными html страницами. В процессе ведётся работа с БД, где хранятся данные пользователей, товаров и заказов, а также категории, точнее их англо-русские соответствия.

Также в приложении реализована работа API, который реализован в отдельных классах, к которым, кстати, и обращается сервер в процессе своей работы.

Используется и файловая система  сервера для хранения фотографий товаров, пользователей и некоторых слайдов.
<>
Из интересных решений можно отметить использование компонентов Bootstrap для декораций страниц, а также использование технологии cookie-файлов для сохранения текущего сеанса, то есть авторизации, и метода сортировки товаров на страницах.
Также хочу заметить, что с помощью API можно выполнять практически любые действия, которые доступны на сайте, от просмотра товаров, до создания заказов.
Забегая вперёд, сразу скажу, что именно этот API будет использоваться в моем следующем проекте.
<>
На этом всё, спасибо за внимание.
